## ğŸ“¦ Installation

### Docker (Recommended)
```bash
git clone https://github.com/JonSnow1807/api-orchestrator.git
```

### Manual Installation
```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn src.main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
